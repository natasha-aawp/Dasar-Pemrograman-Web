Sejarah HTML


Hypertext Markup Language ( HTML ) adalah bahasa markah standar untuk dokumen yang dirancang dan ditampilkan di web browser.

Apa itu CSS


Cascading Style Sheet ( CSS ) merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS merupakan bahasa pemograman.

PHP


( Hypertext Prepocessor ) PHP digunakan untuk merujuk pada bahasa pemrograman rekursif. Secara umum, PHP adalah bahasa pemrograman open source yang digunakan untuk mengembangkan suatu situs website.
